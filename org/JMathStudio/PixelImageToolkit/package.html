This package contain classes defining various types of PixelImage (which are the image standard for this toolkit) and provide many of the useful operations on the same.

Pixel Images

Following set of PixelImages are defined/supported in this package :
  • Colour PixelImage :
    • This family of PixelImages represent the true colour images.
  • UInt PixelImages :
    • This family of PixelImages represent the gray scale intensity image with different depth. Intensity for each pixel position is represented by an unsign integer.
  • Binary PixelImage :
    • This PixelImage represent a boolean/binary image.
